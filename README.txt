Installation:
-------------
To install, simply unzip the files into a directory of your choice.


Known bugs:
-----------
On some computers, appointments will be displayed as if they were UTC time rather than the computer's current timezone.


Missing R1 features:
--------------------
When signing up, patients cannot select a current user as an emergency contact.
When signing up, patients cannot add medical info as the medical-info-related features are set for R2.


Running:
--------
To run, simply double-click on "runserver.bat". The server will then be available at http://127.0.0.1:8000/.
The main admin login is:
	username: admin
	password: admin
